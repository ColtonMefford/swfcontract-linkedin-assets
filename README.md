# swfcontract-linkedin-assets

SWFcontract templates and screened project photos for LinkedIn/Instagram asset generation.
Mirrors `mecho-linkedin-assets`.

- `templates/` — SWFcontract post templates (clean RGB). Note: `swf_template3.jpg` is blank/unusable.
- `projects/` — screened, prepped project photos (no people, no third-party branding).

Raw base: `https://raw.githubusercontent.com/ColtonMefford/swfcontract-linkedin-assets/main/`
